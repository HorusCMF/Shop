Horus CMF

Modern Solution for E-Commerce for Minimalist Operations

Horus CMF is an open source e-commerce solution for PHP, based on the Symfony2 framework.

Ultimate goal of the project is to create a webshop engine, which is user-friendly, loved by developers and has a helpful community.

Horus CMF is constructed from fully decoupled components (bundles in Symfony2 glossary), which means that every feature (products catalog, shipping engine, promotions system...) can be used in any other application.

Documentation is available at docs.Horus CMF.org.
